SMS-assistent BY SMS Integration for WooCommerce. Send message after customer register or change order status

== Description ==

SMS-assistent BY SMS Integration will make you to add SMS features to your WooCommerce store for free. Once configured customers will be notified with an SMS when an Account is registered, Order is placed, Order status is updated etc.

* To configure the module and start sending messages you need to have account on http://sms-assistent.by/

Features:

* Send notification SMS to customers on new orders and account registration
* Send notification SMS to managers on new orders and account registration
* Send notification SMS to customers when order status is changed, like: processed, shipped etc.
* Send notification SMS to managers when order status is changed, like: processed, shipped etc.
* Configure custom templates for each SMS notifications
* Set up your brand name/domain as sender id for the SMS
* Easy Installation steps and friendly interface
* Technical support team to help 24/7

== Installation ==

Upload the SMS-assistent plugin to your blog, activate it, and then configure with your login, password.
